Custom Flight Controller Project. 
1) As a microcontroller I decided to go with the ESP32 Wroom 32 https://www.robotics.org.za/ESP32-DEV
2) CP2104 for USB - UART Communication
3) Flashing Sequence for automatic firmware updates. 
4) USB B/C - will decide on one later. 
This is gonna be an epick project and I am really excited to start this amazing project.
I am still not sure if this is the right controller to choose but many tutorials suggest to do so so who am I to go against them.
5) Motor Connectors for Conecting to the motors.
6) H - bridge Circuit for controlling the motors.
7) 3V3 Voltage Regulator for stablilizing the input voltage to the ESP32.
8) Battery Charger to prevent damage to the battery.
9) Accelerometer (MPU6050) which also contains a Gyroscope, a temperature sensor and a Digital Motion Processor that act as sensors for the controller's brain.
